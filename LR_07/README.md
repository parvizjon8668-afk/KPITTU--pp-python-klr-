# Лабораторная работа №7

## Цель
Освоить соответствующие элементы языка Python: работу с файловой системой, открытие, чтение и запись файлов.

## Задание
Реализовать программу, которая создает текстовый файл, записывает в него данные и выводит содержимое на экран.

## Результат
Рабочая программа на языке Python с комментариями.
