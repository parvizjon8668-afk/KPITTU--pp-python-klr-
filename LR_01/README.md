# Лабораторная работа №1

## Тема
Подготовка рабочего места для командной работы на основе IDE Visual Studio Code
для программирования на языке Python и тестирование программ.

## Цель
Подготовить и настроить среду разработки Visual Studio Code для программирования
на языке Python и апробировать её работоспособность.

## Задание
Установить Python, настроить Visual Studio Code, создать проект и реализовать
программу по индивидуальному заданию.

## Результат
Рабочая программа на языке Python с комментариями.
